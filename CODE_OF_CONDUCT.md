# Contributor Covenant Code of Conduct

This project adopts the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/). By participating in this project, you agree to abide by its terms.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by initiating a private discussion with the project maintainer at [github.com/niclaslindstedt/zig](https://github.com/niclaslindstedt/zig). All complaints will be reviewed and investigated promptly and fairly. All maintainers are obligated to respect the privacy and security of the reporter.
