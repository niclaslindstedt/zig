# Dropbox

Drop any files relevant to your consultation into this folder. The triage nurse
and specialists can read them during the workflow to inform their assessment.

## Examples of useful files

- Lab results (blood work, urinalysis, pathology reports)
- Imaging reports (X-ray, MRI, CT scan findings)
- Referral letters from other doctors
- Discharge summaries from previous hospitalizations
- Prescription lists or pharmacy printouts
- Insurance cards or coverage details
- Symptom diary or pain logs
- Photos of visible symptoms (rashes, swelling, etc.)

Any file format works — the agent will read whatever you place here.
